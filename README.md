# 🐍 Python Homework
Backend Logic Practice • Thực hành Logic Backend
(English – Vietnamese)
## 📘 Overview • Giới thiệu
In Day 2, you continue acting as a Backend Developer, building core logic for a system such as:
- Order processing
- Cost calculation
- User condition checking
- Applying business logic
- Working with lists and loops
- Using loop and writing reusable functions

Đóng vai trò Backend Developer, xây dựng các chức năng xử lý dữ liệu giống như trong hệ thống thực tế:
- Xử lý đơn hàng
- Tính toán chi phí
- Kiểm tra điều kiện người dùng
- Áp dụng logic nghiệp vụ (business logic)
- Làm việc với danh sách dữ liệu
- Tự động hóa bằng vòng lặp và function

## 🎯 Learning Objectives • Mục tiêu học tập
Will understand:
- How to use variables, operators, and conditions
- How to loop through data using for and while
- How to write functions with parameters and return values
- How to apply backend-style business logic
- How to process lists and automate repetitive tasks

Nắm được:
- Cách dùng biến, toán tử, if-else
- Cách dùng vòng lặp for/while để xử lý danh sách
- Cách viết function để tái sử dụng logic
- Tư duy backend: tách logic → xử lý → trả về kết quả
- Áp dụng logic nghiệp vụ vào các bài toán thực tế
